Phase Output Submission Template
Issued by: Project Command
Purpose: To standardize indexing and traceability of all phase outputs.

------------------------------------------------------------
📤 Submission Format (to be posted in Command Chat):

Index Code (Provisional): P[X].M[Y].[TYPE][##]
Phase: X
Module: Y
Type: [DT = Data Table | FG = Figure | TX = Text | RS = Research Summary | SC = Scenario | OT = Other]
Title/Description: [Concise, descriptive name of output]
File Name (if applicable): [e.g., P0_M1_TX01_GlobalFoodProductionSummary.txt]
Storage Location (Folder): 
  • DT → /ssot/phaseX/moduleY/data/
  • TX, FG, RS, SC, OT → /ssot/phaseX/moduleY/outputs/
Source Category: [🟢 A | 🟡 B | 🔴 C] (if data-driven)
Notes: [Optional – methods used, gaps, next steps, etc.]

------------------------------------------------------------
📍 Example:

Index Code (Provisional): P0.M1.FG01
Phase: 0
Module: 1
Type: FG
Title/Description: Pie chart of global plant vs. animal production (calories)
File Name: P0_M1_FG01_PlantVsAnimalProduction.svg
Storage Location: /ssot/phase0/module1/outputs/
Source Category: 🟢 A
Notes: Derived from P0.M1.DT01, visualized using FAOSTAT 2022 data.

------------------------------------------------------------
All phase teams must report each new output using this format.
Project Command will validate and log the entry into the Master Index Register.
